Privacy Policy

Mass Downloader is a Chrome extension designed to assist users in downloading videos and files from supported websites using a desktop companion app.

Data Collection

This extension does not collect, store, or transmit any personally identifiable information (PII).

Permissions Usage

  • Cookies: Only used to forward cookies from the current tab to the local downloader app to support authenticated downloads.
  • Download access: Only used to detect download events for redirection to the app.
  • Host access: Required to detect videos on supported websites such as YouTube, Facebook, TikTok, Vimeo, etc.
  • Storage: Used internally to store temporary user preferences (e.g., enabling/disabling button on video pages).

Data Sharing

No data is sent to third-party servers. All communication is strictly local between the extension and the Mass Downloader desktop app running on your computer.

Analytics & Tracking

This extension does not use any analytics, trackers, or logging mechanisms.
